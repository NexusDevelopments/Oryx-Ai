# Oryx Web

Minimal, high-contrast, grayscale-only web interface for Oryx — VS Code AI Agent.

## Features
- Black, white, and grayscale-only aesthetic
- High-contrast, minimal UI
- Chat interface inspired by Grok and ChatGPT
- No bright colors or decorative elements
- Production-ready Next.js/React code

## Getting Started

```sh
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
